RULES OF THUMB
--------------
Never touch: SD0, SD1, SD2, SD3, CLK, CMD (GPIO6 to GPIO11, wired to the module flash).
Input only (no outputs, no pull ups): SVP (36), SVN (39), P34, P35.
Strapping pins (state at boot matters): P0, P2, P5, P12, P15.
ADC2 pins (25, 26, 27, 14, 12, 13, 4, 2, 15) cannot do analogRead while Wi-Fi is on; use ADC1 (32, 33, 34, 35, 36, 39) for analog sensors.

VITAL WIRING PLAN
--------------------
PZEM-004T v3 (energy meter, UART):
  PZEM TX  -> P16 (RX2)
  PZEM RX  -> P17 (TX2)
  PZEM 5V  -> 5V
  PZEM GND -> GND
  Note: PZEM TX idles at 5V on some modules; use a level shifter or divider into P16.

DS18B20 (temperature):
  Yellow (DATA) -> P32
  Red (VCC)     -> 3V3
  Black (GND)   -> GND
  Note: internal pull up on P32 (INPUT_PULLUP) drives the bus, no external resistor needed.

16x2 I2C LCD (PCF8574 backpack):
  VCC -> 5V
  GND -> GND
  SDA -> P13
  SCL -> P14
  Note: SDA/SCL are custom I2C pins set via Wire.begin(13, 14), chosen so the four
        wires cluster next to the left-strip 5V and GND terminals for short runs.
  Note: hd44780 auto-detects the backpack I2C address and register mapping and
        drives whichever answers, so no address jumper guessing is needed.
